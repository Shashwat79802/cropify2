Create network:
docker network create cropify-network


Create mysql database:
1. docker build . -t mysql_db
2. docker run -p 3306:3306 --network cropify-network --name cropify3 mysql_db


If laptop is restarted, run this to start database:
docker start cropify3

To run the app:
1. docker compose build
2. docker compose up


To stop the app:
1. press ctrl+c
2. docker compose down
